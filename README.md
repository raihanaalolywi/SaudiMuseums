# Musera

---

## 📌 Project Description
This project is a web platform designed to gather all cultural authorities and museums in the Kingdom of Saudi Arabia in one place.  
Each cultural or governmental authority can register on the platform and add its own information along with the museums it manages.

Visitors can browse these authorities, explore their museums, view museum details, and book visit tickets.  
The platform provides a simple and user-friendly experience, allowing users to:

- View all registered cultural authorities  
- Explore museums under each authority  
- Access detailed museum information  
- Book visit tickets for museums  

Authorities have additional privileges:

- Log into their authority account  
- Add or update authority information  
- Add new museums under their management  

---

## 📌 Features
- User Authentication (Register, Login, Logout)
- Profile page with avatar image and visited museums count
- Dynamic museum pages (details + images + description)
- Favorites system (users can like/save museums)
- Authorities (cultural organizations) listing + detail pages
- Responsive design using a CSS framework (e.g., Bootstrap)
- Template inheritance and organized project structure
- Use of Static & Media files
- Python-dotenv for sensitive data management
- Pagination, Search & Filtering for museums
- Git & GitHub version control
